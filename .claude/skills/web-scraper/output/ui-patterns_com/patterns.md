# Design patterns


 
 
 
 
 
 
 
 
 

 
 
 
 
 
 

 


 
 
 


 
 
 
 
 User Interface Design patterns are recurring solutions that solve 
 common design problems. Design patterns are standard reference points 
 for the experienced user interface designer.


 Persuasive design patterns are recurrent, psychology-based strategies 
 that deliberately steer users toward intended outcomes. They provide 
 designers with a systematic framework for embedding purposeful influence 
 in an interface, enabling clear alignment between organisational objectives 
 and user value while maintaining transparency and ethical responsibility.


 Design patterns provide a common language between designers. They allow for debate
 over alternatives, where merely mentioning the name of a design pattern implicitly
 carries much more meaning than merely the name.


 Dwell into the patterns below to learn a common language of web design.
